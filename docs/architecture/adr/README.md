# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Insight Monitor project.

## What is an ADR?

An ADR documents a significant architectural decision, including the context, options considered, and the chosen approach. ADRs serve as a historical record of why the system is built the way it is.

## Template

```markdown
# ADR-NNNN: Title

## Status

[proposed | accepted | deprecated | superseded]

## Context

What is the issue motivating this decision?

## Decision

What is the change being made?

## Consequences

What becomes easier or harder after this change?
```
